# student_grade_predict
Первый проект по машинному обучению

# Прогнозирование итоговых оценок студентов

## Цель проекта
Разработать модель линейной регрессии для прогнозирования итоговой оценки студентов по математике на основе демографических, социальных и академических характеристик.

## Использованные технологии
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

## Краткое описание данных
Данные содержат информацию об успеваемости 395 учеников двух португальских школ за 2005-2006 учебный год, включая:
Демографические характеристики (школа, пол, возраст, адрес)
Семейные факторы (образование родителей, работа, размер семьи)
Академические показатели (время на учёбу, пропуски, предыдущие оценки)
Социальные аспекты (свободное время, отношения с семьей, здоровье)

## Ключевые выводы и результаты
Проведена предобработка данных с удалением аномальных записей (студенты с нулевой итоговой оценкой)
Визуализировано распределение оценок за первый и третий периоды обучения
Построена модель линейной регрессии для прогнозирования итоговой оценки G3
Определены наиболее значимые факторы, влияющие на успеваемость студентов
Модель может быть использована для выявления студентов, нуждающихся в дополнительной поддержке
